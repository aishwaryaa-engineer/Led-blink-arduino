# SOS Morse Code - Arduino

## What it does
Blinks an LED in SOS morse code pattern using Arduino Uno

## Components
- Arduino Uno
- LED
- 220 ohm resistor

## My Logic
- Short blink = 200ms (S)
- Long blink = 2000ms (O)
- Pause between letters = 500ms
- Final pause = 1000ms

## What I learned
- GPIO programming
- Timing control in embedded systems
- Morse code logic implementation
